# Projet Nuxt 4

Ce dépôt contient un projet Nuxt 4 configuré avec la structure `app/` standard.

## Structure Nuxt 4

Le projet utilise la structure suivante :

- `app/` : point d’entrée principal de l’application
- `app/assets/` : actifs traités par Nuxt
- `app/components/` : composants Vue réutilisables
- `app/composables/` : fonctions composables
- `app/layouts/` : layouts partagés
- `app/middleware/` : middleware avant navigation
- `app/pages/` : pages basées sur les routes
- `app/plugins/` : plugins Nuxt/Vue
- `app/utils/` : utilitaires partagés

## Commandes

- `pnpm install`
- `pnpm dev` pour lancer le serveur de développement
- `pnpm build` pour construire l'application
- `pnpm preview` pour prévisualiser le build
